Removes shortcodes from Yoast meta descriptions. Adds clean meta descriptions if none exist.

== Description ==

This plugin cleans meta descriptions of any shortcode that you input. If no Yoast meta description exists, this plugin will insert one without the specified shortcodes. This is NOT an official Yoast plugin/addon.
